Typography

The Distinctive Theme starts with great Typography. It includes the Free Web Font Montserrat for clear clean text. Wells have also been extended to include an optional raised state to give focus to the most important content.

Progress bars

The Distinctive Theme contains several enhancements to Twitter Bootstrap's default progress bars. They are now framed in a small well, enhancing their look. Additionally, they now come in several new colors that match the theme.

Forms

Forms are one of the most important parts of a website and this is one area where Twitter Bootstrap shines. The Distinctive Theme enhances forms as raised wells are often a great place for a form. It also includes the File Upload Plugin and styling from Jasny's excellent work which creates a file upload input which is consistent cross-browser and much more visually appealing. Vertical forms are also enhanced with slightly different control labels and better spacing.

Miscellaneous

Twitter Bootstrap is truly a UI toolkit with lots of additional components, many of which have been enhanced in the Distinctive theme. Of particular note is Pagination which has been greatly improved. It is now wrapped in a small well and with the help of some tasteful shadows it has much more definition and personality.
